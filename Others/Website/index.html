Let's Go by Nagoya Municipal Subway!!

「もっと! 地下鉄 大名古屋」は、オープンソースの3DCG地下鉄運転ゲームです。

スクリーンショット


動作環境

OS
Windows 10 / 11
CPU
Intelまたは同等の互換プロセッサ 2GHz以上 (マルチコアを推奨)
メモリ
8GB以上 (16GB以上を推奨)
ディスプレイ
1366x768ピクセル以上の解像度で色深度 32bit True color以上表示可能なもの
ストレージ
100MB以上の空き容量
グラフィックデバイス
DirectX 9以降に対応したグラフィックデバイス
サウンドデバイス
DirectSoundに対応したサウンドデバイス
